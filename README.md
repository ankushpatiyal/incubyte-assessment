# incubyte-assessment
## Pre-requisites

* Rbenv or Rvm

## Steps

### Install Ruby

```
    rbenv install 3.1.0
```

OR

```
    rvm install 3.1.0
```


### Use

```
    rbenv local 3.1.0
```

OR

```
    rvm use 3.1.0
```

### Install Bundler
```
    gem install bundler
```

### Install depedencies

```
    bundle Install
```
